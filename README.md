# 观鸟雷达

一个用于查看近期观鸟热点地点和鸟种列表的静态网页。

## 发布到 GitHub Pages

1. 打开 https://github.com/new
2. Repository name 填：birdwatching-radar
3. 选择 Public
4. 点击 Create repository
5. 在仓库页面点击 uploading an existing file
6. 上传本文件夹里的 index.html、.nojekyll、README.md
7. 点击 Commit changes
8. 进入 Settings -> Pages
9. Source 选择 Deploy from a branch
10. Branch 选择 main，文件夹选择 /root
11. 点击 Save

几分钟后，GitHub 会给出一个公开链接，通常类似：
https://你的用户名.github.io/birdwatching-radar/

注意：eBird API Token 只在浏览器页面里输入，不会保存在这个仓库文件中。
